# Industrial Telemetry & Weight Visualizer

A cross-platform mobile application developed with **React Native** (Expo) designed to connect, process, and visualize real-time dynamic data streams from industrial weighing sensors and transmitters via wireless communication protocols.

## 🚀 Key Features
* **Real-Time Data Streaming:** Visualizes live weight measurements and status flags with low latency.
* **Data Processing Logic:** Implements conditional state management to interpret raw multi-channel telemetry packages.
* **Connection Management:** Efficiently handles continuous data input streams without performance drops.

## 🛠️ Tech Stack
* **Frontend:** React Native, JavaScript (ES6+), Expo
* **State Management:** React Hooks (`useState`, `useEffect`, `useLocalSearchParams`)
* **Hardware Integration:** BLE (Bluetooth Low Energy) / Serial communication logic prepared for microcontroller environments.
